# CNC AI 優化器 Vue.js 模板完整解析

## 概述
這是一個完整的 CNC（電腦數值控制）AI 優化平台的前端應用程式，使用 Vue.js 3 Composition API 構建。

## 主要功能模組

### 1. 應用程式狀態管理 [app.vue: 1575-1590]
- currentView: 控制當前顯示的視圖（登陸頁/登入/主應用） [app.vue: 1575]
- selectedPlatform: 選擇的平台版本（v0/v1/data） [app.vue: 1576]
- activeMode: 當前活動模式（new/iterate/simulation） [app.vue: 1577]
- showSimulationConfig: 控制模擬配置顯示 [app.vue: 1578]

### 2. 視圖結構

#### 2.1 登陸頁面 (Landing Page) [app.vue: 3-67]
- 功能: 平台選擇入口
- 三個主要選項: v0基礎版、v1進階版、資料上傳 [app.vue: 19-67]
- 設計特點: 使用網格佈局，每個選項都有圖標和懸停效果

#### 2.2 登入頁面 (Login Screen) [app.vue: 69-108]
- 功能: 使用者認證
- 表單元素: 使用者名稱和密碼輸入 [app.vue: 85-100]
- 驗證: 基本的表單驗證邏輯 [app.vue: 1591-1597]

#### 2.3 主應用程式 (Main Application) [app.vue: 110-1573]
- 側邊欄: 包含應用程式標題、導航選單和登出按鈕 [app.vue: 112-158]
- 主要內容區域: 根據不同模式顯示相應的界面 [app.vue: 160-1573]

### 3. 三種主要工作模式

#### 3.1 新專案生成模式 (New Project Generation) [app.vue: 160-300]
- 功能: 建立全新的 CNC 優化專案
- 流程: 選擇專案 → 選擇機台 → 命名專案 → 配置參數
- 特色: 提供專案選擇模態框 [app.vue: 1500-1573]

#### 3.2 迭代先前專案模式 (Iterate Previous Projects) [app.vue: 302-350]
- 功能: 基於現有專案進行優化和調整
- 流程: 選擇現有專案 → 選擇迭代類型 → 調整參數 [app.vue: 302-350]
- 迭代類型: 參數調整、材料變更、刀具修改、策略優化 [app.vue: 1550-1573]

#### 3.3 模擬建立模式 (Simulation Creation) [app.vue: 160-300]
- 功能: 建立和執行 CNC 模擬
- 特色: 模擬執行表格、檔案上傳、範本下載 [app.vue: 160-300]
- 狀態管理: 追蹤模擬的執行狀態

### 4. 聊天界面系統 [app.vue: 352-450]

#### 4.1 消息結構 [app.vue: 352-450]
- id: 唯一標識符
- type: 消息類型（user/assistant）
- content: 消息內容（支援 HTML）
- showFileUpload: 是否顯示檔案上傳界面
- showConfig: 是否顯示配置界面
- showResults: 是否顯示結果界面

#### 4.2 智能回應系統 [app.vue: 1450-1480]
- 基於關鍵字的智能回應
- 支援修改、解釋、比較、模擬等關鍵詞
- 動態生成回應內容
